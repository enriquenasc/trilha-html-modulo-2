Exemplos de Tags de Mídias no HTML

Tag IMG

A tag <img> é usada para incorporar imagens em uma página.

Tag AUDIO

A tag <audio> é usada para reproduzir arquivos de áudio.

Tag VIDEO

A tag <video> é usada para reproduzir arquivos de vídeo.

Tag TRACK

A tag <track> é usada para adicionar legendas ou descrições a vídeos.

Tag IFRAME

A tag <iframe> é usada para incorporar conteúdo de outras páginas, como vídeos do YouTube.
